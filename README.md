## Accessibility

This site is built with accessibility as a first-class concern, not an afterthought:

- Semantic HTML landmarks (`<header>`, `<main>`, `<address>`, `<section>`) with `aria-labelledby` linking every section to its heading
- Visible, high-contrast focus states for full keyboard navigation
- A "skip to main content" link for screen reader and keyboard users
- Decorative icons marked `aria-hidden`, with descriptive text for meaningful links
- Print-optimized stylesheet for a clean, one-click PDF export of the CV view
